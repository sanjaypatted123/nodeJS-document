## âœï¸ Create a Template (views/index.ejs)

```html
<!DOCTYPE html>
<html>
<head>
  <title><%= title %></title>
</head>
<body>
  <h1>Welcome, <%= name %>!</h1>
</body>
</html>
```

> `<%= %>` is used to output escaped data.
> `<%- %>` outputs unescaped HTML.
> `<% %>` allows plain JS (e.g., loops, if-else).
